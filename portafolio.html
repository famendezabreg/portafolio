Desarrollo Web I

Descripción del Curso

La materia Desarrollo Web I se centra en la creación de sitios web utilizando HTML, CSS y JavaScript. Los estudiantes aprenderemos a diseñar y desarrollar páginas web responsivas y accesibles.

Contenidos Principales

  • HTML5 y CSS3
  • Diseño Responsive
  • Flexbox y Grid
  • Media Queries
  • Formularios HTML
  • Buenas prácticas de desarrollo web

Guías de Ejercicios

Guía 1: Introducción a HTML

Se centra en el lenguaje de marcado que define el contenido y la organización de una página web (texto, imágenes, enlaces, listas, etc.). Es la base de todo sitio.

Ver ejercicios

Guía 2: CSS Básico

Se enfoca en el lenguaje de estilos que se utiliza para darle apariencia (colores, fuentes, fondos, márgenes) y diseño (diseños flexibles, responsividad) a la estructura creada con HTML.

Ver ejercicios

Guía 3: JavaScript, la Interactividad de la Web

Introduce el lenguaje de programación que convierte una página estática en una aplicación dinámica. Permite añadir comportamiento, manipular la interfaz y responder a las acciones del usuario

Ver ejercicios

Guía 4: Responsive Design

Diseño adaptativo con media queries y viewport para diferentes dispositivos.
